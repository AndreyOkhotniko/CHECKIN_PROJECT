# Чекин Desktop — Дизайн-система

> Веб-интерфейс приложения «Чекин» для компьютера. Адаптация мобильной книги штампов под десктоп: верхнее меню, сетки карточек, резиновая ширина. Отдельного приложения нет — это веб.

Базовый визуальный язык (цвета, штампы, типографика Inter) наследуется из мобильной версии — см. `design-system.md`. Здесь описаны **отличия и паттерны десктопа**.

---

## Ключевые решения адаптации

| Аспект | Мобайл | Десктоп |
|--------|--------|---------|
| Навигация | Нижний таб-бар | **Верхнее меню (topbar)**, sticky |
| Список мест | Вертикальный список | **Сетка карточек** (auto-fill) |
| Детали места | Один столбец, скролл | **Two-column**: штамп-панель (sticky) + контент |
| Формы | Полноэкранные шаги | **Two-column**: поля + живое превью справа |
| QR-сканер | Экран камеры | **Удалён** — чек-ин только через телефон |
| Ширина | 402px фикс | **Резиновая**, max-width 1320px |
| Удаление | Bottom sheet | **Центрированная модалка** |

Сканирование на десктопе невозможно (нет камеры в руках), поэтому на странице места показывается инструкция «отсканируйте QR через мобильное приложение».

## Сетка и контейнеры

```css
.dwrap{ max-width:1320px; margin:0 auto; padding:40px 32px 64px; }
.dtopbar-in{ max-width:1320px; height:64px; padding:0 32px; }
```

- **Контейнер:** центрированный, max-width 1320px, резиновый до этого предела
- **Сетка карточек:** `repeat(auto-fill, minmax(280px, 1fr))`, gap 18px
- **Детали / формы:** CSS Grid two-column, правая колонка `position:sticky; top:96px`

### Брейкпоинты
| Ширина | Поведение |
|--------|-----------|
| > 900px | Полный two-column, topbar с навигацией |
| ≤ 900px | Детали/формы → один столбец, auth скрывает брендовую панель |
| ≤ 560px | Скрыта nav в topbar, уменьшены отступы, кнопки без подписей |

## Топбар

```css
.dtopbar{ position:sticky; top:0; background:rgba(247,244,239,.85); backdrop-filter:blur(16px); border-bottom:1px solid #E8E2DB; }
```

Структура: `[Логотип ЧЕКИН] [Навигация] [Действия + Аватар]`

- **Гость:** Места · Коллекция
- **Владелец:** Мои места · Аналитика + кнопка «+ Добавить место» + бейдж «для бизнеса» у лого
- Активная ссылка: `background:rgba(196,81,58,.08); color:#1A1714; font-weight:600`
- Аватар: круг 40px, тёмный фон, инициалы

## Десктоп-акценты (новое относительно мобайла)

1. **Hover-состояния карточек** — `translateY(-3px)` + усиленная тень
2. **Split-экран входа** — тёмная брендовая панель слева с коллажем из 6 штампов + форма справа
3. **Живое превью в формах** — справа sticky-панель показывает карточку/штамп в реальном времени
4. **Богатые строки в дашборде** — штамп + адрес + мини-статы (всего / сегодня) + мини-QR + действия в одной строке
5. **Расширенная аналитика** — 4 стат-карточки в ряд, крупный бар-чарт + лента активности рядом (two-column)

| Класс | Назначение |
|-------|-----------|
| `.dbtn` | Primary, терракот, padding 13×24 |
| `.dbtn-ghost` | Вторичная, бордюр |

### Поля
```css
.dinput{ padding:13px 16px; border:1.5px solid #E8E2DB; border-radius:12px; }
.dinput:focus{ border-color:#C4513A; }
.dsearch{ max-width:420px; }  /* поиск с иконкой-лупой слева */
```

### Карточки
```css
.dcard{ background:#fff; border:1px solid #EFEAE3; border-radius:18px; box-shadow:0 1px 3px rgba(0,0,0,.04); }
.dcard.place:hover{ transform:translateY(-3px); box-shadow:0 10px 28px rgba(0,0,0,.09); }
```

### Стат-карточка
```css
.dstat{ padding:22px 24px; border-radius:16px; }
.dstat-val{ font:700 32px Inter; }  /* крупное число */
.dstat-lbl{ font:400 14px Inter; color:#8A7B70; }
```

### Чипсы (фильтры)
```css
.dchip{ padding:8px 16px; border-radius:22px; background:#EDE9E3; color:#8A7B70; }

```

### Модалка
```css

```

## Навигационные потоки

```
Вход (выбор роли)
  ├─ Гость → Места → Место → Штамп получен → Коллекция → Карточка штампа
  │                    Профиль ←→ (апселл: стать владельцем)
  └─ Владелец → Дашборд → Аналитика
                   ├─ Добавить → Дизайн штампа → QR-код
                   ├─ Редактировать (предзаполнено)
                   └─ Удалить (модалка)
```
