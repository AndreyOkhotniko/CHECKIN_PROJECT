## TL;DR за 60 секунд

- Это **бэкенд на FastAPI** для приложения «чек-инов»: пользователь сканирует
  QR-код места и получает «штамп» (как печать в коллекционной карточке).
- Стек: **FastAPI + SQLAlchemy 2.0 (async) + PostgreSQL + Alembic + JWT**.
- Архитектура — классическая слоёная: `endpoints → schemas → models → db`.
- Аутентификация — **JWT** (access + refresh токены), пароли хешируются `bcrypt`.
- Состояние: ядро API реализовано (auth, профиль, места, штампы, категории),
  но **миграций ещё нет** и **тесты отсутствуют**. Подробности — в файле 05.
